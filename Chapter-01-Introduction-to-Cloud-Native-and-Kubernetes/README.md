# Chapter 1 – Introduction to Cloud Native and Kubernetes

This folder contains **reference-only YAML and configuration snippets**
aligned with Chapter 1 of the book *Ultimate Kubernetes for Cloud-Native Applications*.

These examples support conceptual understanding of:
- Containers and Docker
- Kubernetes core objects
- Scaling and autoscaling
- Probes and self-healing
- RBAC and security
- CI/CD and DevOps integration

⚠️ These files are **NOT production-ready**
and are **NOT intended to be executed directly**.
