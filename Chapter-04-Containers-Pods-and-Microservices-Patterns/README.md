# Chapter 4 – Containers, Pods, and Microservices Patterns

This folder contains **reference-only Kubernetes YAML and Docker snippets**
used in Chapter 4 of the book.

These examples demonstrate:
- StatefulSets
- DaemonSets
- Static Pods
- API Gateway pattern
- gRPC services
- Containerized ETL pipelines

⚠️ These snippets are **not production-ready** and are provided strictly
for conceptual understanding aligned with the book content.
