# Chapter 2 – Setting Up Kubernetes Clusters

This directory contains **reference-only configuration files and commands**
aligned with Chapter 2 of the book *Ultimate Kubernetes for Cloud-Native Applications*.

The snippets cover:
- Kubernetes installation methods (kubeadm, Minikube, Kind)
- Managed Kubernetes on AWS, Azure, and GCP
- Cluster networking and storage basics
- Troubleshooting cluster and node issues
- Security best practices
- Designing resilient Kubernetes-based data pipelines

⚠️ These examples are for **conceptual understanding only**  
and are **not production-ready or fully executable**.
